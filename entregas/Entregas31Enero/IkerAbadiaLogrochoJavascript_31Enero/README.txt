PASOS A SEGUIR PARA QUE FUNCIONE EL PROYECTO
1- IMPORTAR LA BASE DE DATOS EN PHPMYADMIN
2- METER LA CARPETA "logrocho" DIRECTAMENTE EN HTDOCS
3- CAMBIAR LOS DATOS DB_INFO DB_USER Y DB_PASS DEL ARCHIVO BD.PHP DENTRO DE LA CARPETA MODEL Y PONER LOS DE TU CONEXION A LA BASE DE DATOS
4- LOS DATOS DEL USUARIO ADMIN PARA EL LOGIN SON user:admin password:admin
